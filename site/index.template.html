📌 กดติดตามปฏิทิน

iPhone / iPad / Mac — กดแล้วยืนยัน "สมัครรับ" Google Calendar (Android) — กดแล้วเลือก "เพิ่ม"
Outlook — ขั้นตอนเพิ่มเอง
  1. เปิด Outlook Calendar แล้วกด Add calendar
  2. เลือก Subscribe from web
  3. วางลิงก์ด้านล่างนี้ ตั้งชื่อปฏิทิน แล้วกด Import

หรือคัดลอกลิงก์ไปวางเองในแอปปฏิทินใดก็ได้ (แตะที่ลิงก์เพื่อเลือกทั้งบรรทัด): {{ICS_URL}}

❓ วิธีติดตาม ทีละขั้น

iPhone / iPad / Mac
  • แตะปุ่ม iPhone / iPad / Mac ด้านบน
  • เครื่องจะเด้งหน้าต่าง "สมัครรับปฏิทิน" — กด สมัครรับ / Subscribe
  • เสร็จแล้ว — กำหนดส่งใหม่ ๆ จะตามมาเองในแอปปฏิทินเดิม
Android / Google Calendar
  • แตะปุ่ม Google Calendar (Android) ด้านบน แล้วล็อกอินบัญชี Google (ถ้ายังไม่ได้ล็อกอิน)
  • กด เพิ่ม / Add ในหน้าที่เปิดขึ้นมา
  • เปิดแอป Google Calendar ในมือถือ — ปฏิทินใหม่จะโผล่ในรายการปฏิทินอัตโนมัติ
Outlook
  • Add calendar → Subscribe from web
  • วางลิงก์จากช่อง "คัดลอกลิงก์" ด้านบน แล้วกด Import

ℹ️ ควรรู้ก่อนใช้

⏱ Google Calendar รีเฟรชปฏิทินแบบติดตามประมาณวันละครั้ง — เหมาะกับ deadline ที่รู้ล่วงหน้า ไม่เหมาะกับประกาศด่วนรายชั่วโมง และแนะนำให้ตั้ง default reminder ของปฏิทินตัวเองด้วย เพราะบางแอปไม่รับการแจ้งเตือนที่ฝังมากับ feed

🔒 ปฏิทินนี้เป็น feed อ่านอย่างเดียว ผู้จัดทำไม่เห็นและไม่เก็บข้อมูลว่าใครติดตาม ไม่มีบัญชี ไม่มีการล็อกอิน ไม่มี analytics
